# Variables d'environnement

- `DATABASE_URL` : URL PostgreSQL Prisma
- `APP_BASE_URL` : URL publique de l'application
- `AUTH_SECRET` : secret pour les sessions
- `DEFAULT_TIMEZONE` : timezone par défaut des nouveaux foyers
- `OCCURRENCE_PAST_DAYS` : fenêtre passée à conserver pour la génération
- `OCCURRENCE_FUTURE_DAYS` : horizon futur généré
