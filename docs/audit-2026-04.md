# Audit 2026-04 — Plan de simplification

État du chantier de simplification post-audit. Source : audit du 26 avril 2026 sur la
base `main` (~18k lignes app, 148 fichiers TS/TSX, 48 routes API, 16 modèles Prisma).

## Constats principaux

- 26 routes API dupliquent le même prologue `requireUser` + `householdMember.findFirst`.
- 3 patterns de soumission de formulaires coexistent (`<form action="/api/...">`,
  `<ClientForm>`, `useFormAction`).
- 5 composants ≥ 400 lignes mélangent layout / state / IO.
- `globals.css` (811 lignes) contient ~30 lignes d'overrides `!important` pour le dark
  mode, à remplacer par `@variant dark`.
- Dépendance `@hookform/resolvers` déclarée mais non importée.
- Pas d'usage de Server Actions ni de `useOptimistic` (React 19 / Next 16).
